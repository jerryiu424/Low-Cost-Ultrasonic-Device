\subsection{Detailed Description}
The \hyperlink{classCube}{Cube} class. 

This class is a child class of the \hyperlink{classContainer}{Container} class. Since the shape of this \hyperlink{classContainer}{Container} is a cube the only measurement we require is side length. From this we can determine the max possible volume of the container.
