\subsection{Detailed Description}
The \hyperlink{classCuboid}{Cuboid} class. 

This class is a child class of the \hyperlink{classContainer}{Container} class. Since the shape of this \hyperlink{classContainer}{Container} is a cuboid the dimensions are width, length and height. From this we can determine the max possible volume of the container.
