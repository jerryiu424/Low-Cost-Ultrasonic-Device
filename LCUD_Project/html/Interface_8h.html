The Interface program.

This program starts by either loading a pre-existing container, or creates a new one; it then reads data from the sensor to determind the volume of the fluid in the container. We then periodically take an average of the data, and send it to our database.

void CubeFunc ( Cube  c)

Calculates volume data from sensor information and sends average to the database.

This function takes in a Cube object, uses the sensor connected to the container to get the duration and temperature. From there we calculate the level of the fluid and the volume. Every 10 times the information is calculated, the function averages this data, creates an SQL query and sends the query to the database.

Parameters
cThe Cube container that will be measured
